# 知书：仿照简书实现的一个阅读社区

## 功能介绍

* 文章相关
    * 评论、收藏、点赞文章
    * 富文本编辑器
    * 推荐文章
* 消息中心
    * 评论
    * 投稿请求
    * 喜欢和赞
    * 关注
    * 其他提醒
* 关注中心
    * 关注
    * 粉丝
* 个人中心
* 搜索
    * 文集搜索
    * 文章搜索
    * 专题搜索
    * 用户搜索
    * 最近搜索
* 文章用户排行榜
